# Open Source [Next.js](https://nextjs.org) Portofolio Web Template

## Stack : 
- NextJs (typescript)
- TailwindCss

## Getting Started

```bash
git clone https://github.com/ezhasyafaat/ezhasyafaat.me.git

mv ezhasyafaat.me yourname

cd yourname

yarn dev
# or
npm run dev
```
